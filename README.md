1. What is a variable?

Answer:
A variable is a named memory location used to store data that can be changed during program execution.

2. What are data types in Python?

Answer:
Data types define the type of value stored in a variable.

Examples:

int
float
str
bool
list
tuple
set
dict
3. What is dynamic typing?

Answer:
Python automatically determines the data type of a variable at runtime, so you don't need to declare it explicitly.

Example:

x = 10
x = "Hello"
4. How do you check the data type of a variable?

Answer:
Using the type() function.

Example:

x = 10
print(type(x))
5. What is the difference between = and ==?

Answer:

= is the assignment operator.
== is the comparison operator.
6. What is the input() function?

Answer:
The input() function takes input from the user as a string.

Example:

name = input("Enter name: ")
7. What is the print() function?

Answer:
The print() function displays output on the screen.

Example:

print("Hello")

9. Is Python case-sensitive?

Answer:
Yes. name, Name, and NAME are three different variables.

10. What are the rules for naming variables?
Must start with a letter or _
Cannot start with a number
Cannot contain spaces
Cannot use Python keywords (like if, for, while)
Use meaningful names

Example:

student_name = "Priyanka"   # ✅
2name = "Priyanka"          # ❌
